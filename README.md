# Filament Website

Website based on filament

Full Documentation : [_docs/index.md](_docs/index.md)

## Requirement

- Laravel 12
- PHP 8.4
- MySQL

## Features

- User Management
  - Multiple User with Multiple Roles : Admin, Editor, Contributor, Guest
- Settings Page
  - Configuration Management
- Post & Page Management
  - Include revisions
  - multiple languages
  - publish date
  - review system
- Page Management
  - Include revisions
  - multiple languages
  - publish date
  - review
- Templating
